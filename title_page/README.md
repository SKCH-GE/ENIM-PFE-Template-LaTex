# Title Page Assets

This folder stores the external title-page files supplied with the template, such as the official PDF cover and editable `.docx` variants.

The main LaTeX entry point includes `pgdg-en.pdf` as the default cover page. Edit the matching `.docx` file, export it to PDF, and replace the default PDF when preparing a final report. Use `pgdg-fr.pdf` instead in `../main.tex` if the French cover is required.

Keep this folder for title-page source files only. Do not place chapter figures or report screenshots here.
