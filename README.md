# LaTeX Report Workspace

This folder contains the LaTeX source for the final report. `main.tex` is the compilation entry point and includes the front matter, chapters, appendices, and bibliography.

## Contents

- `main.tex`: document preamble, metadata, package setup, and include order.
- `frontmatter/`: dedication, acknowledgements, abstract, and nomenclature.
- `chapters/`: numbered thesis chapters from introduction to conclusion.
- `appendices/`: supplementary material and additional results.
- `bibliography/`: reference section source.
- `images/`: empty figure folder structure for student-supplied media.
- `title_page/`: external title-page PDF and editable document files.

Files such as `.aux`, `.toc`, `.lof`, `.lot`, `.fls`, `.fdb_latexmk`, `.synctex.gz`, and `main.pdf` are generated by LaTeX builds and should not be treated as source text.
